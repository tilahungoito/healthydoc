# Screenshots Directory

This directory contains screenshots of the AI Health Assistant application for the README documentation.

## How to Add Screenshots

### Step 1: Take Screenshots
1. Run your application locally: `npm run dev`
2. Navigate to different pages/features
3. Take screenshots using:
   - **Windows**: `Win + Shift + S` (Snipping Tool) or `PrtScn`
   - **Mac**: `Cmd + Shift + 4`
   - **Browser**: Right-click → Inspect → Device Toolbar → Screenshot

### Step 2: Save Screenshots
1. Save screenshots in this directory (`public/screenshots/`)
2. Use descriptive filenames:
   - `welcome-page.png`
   - `dashboard.png`
   - `health-analysis.png`
   - `ai-doctor.png`
   - `malaria-detection.png`
   - `pneumonia-detection.png`
   - `body-scanner.png`
   - `health-history.png`
   - `facilities-finder.png`
   - `settings.png`

### Step 3: Image Recommendations
- **Format**: PNG (preferred) or JPG
- **Width**: 1200-1920px (for best display on GitHub)
- **File Size**: Keep under 1MB per image (compress if needed)
- **Aspect Ratio**: 16:9 or 4:3 works well

### Step 4: Update README
The screenshots are already referenced in the main README.md file. Just replace the placeholder images with your actual screenshots using the same filenames, or update the README with your custom filenames.

### Example Markdown Syntax
```markdown
![Description](./public/screenshots/filename.png)
```

### Tools for Image Optimization
- [TinyPNG](https://tinypng.com/) - Compress PNG/JPG images
- [Squoosh](https://squoosh.app/) - Image compression tool
- [ImageOptim](https://imageoptim.com/) - Batch image optimization
